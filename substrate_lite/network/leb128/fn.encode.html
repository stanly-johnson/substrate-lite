pub fn encode(
    value: impl Into<u64>
) -> impl ExactSizeIterator<Item = u8> + Clone

Returns an LEB128-encoded integer as a list of bytes.

This function accepts as parameter an Into<u64>. As such, one can also pass a u8, u16, or u32 for example. Use encode_usize for the usize equivalent.

Example

use substrate_lite::network::leb128;

assert_eq!(leb128::encode(0u64).collect::<Vec<_>>(), &[0]);
assert_eq!(
    leb128::encode(0x123456789abcdefu64).collect::<Vec<_>>(),
    &[239, 155, 175, 205, 248, 172, 209, 145, 1]
);
